# 八字命理分析应用

## 功能特性

- **八字命理分析**: 基于公历日期自动转换为八字并进行深度分析
- **MBTI性格判定**: 通过能量分布计算16种人格类型
- **五行能量可视化**: 直观的进度条展示五行能量分布
- **格局与用神分析**: 详细的命理格局和调候用神建议

## 设计哲学

- **Muji × 水墨留白**: 极简设计，拒绝厚重装饰
- **宣纸色调**: 背景使用极淡米灰色 (#faf9f6)
- **墨色文字**: 使用深炭灰色 (#2c2c2c)
- **极致留白**: 大量空白空间营造宁静氛围
- **书法字体**: Serif标题配Sans正文

## 技术栈

- **Next.js 16**: React框架
- **TypeScript**: 类型安全
- **Tailwind CSS**: 原子化CSS
- **Framer Motion**: 流畅动画效果
- **Lucide Icons**: 精美图标
- **lunar-typescript**: 八字转换库

## 安装依赖

```bash
npm install framer-motion lucide-react
```

## 运行应用

```bash
npm run dev
```

## 使用说明

1. 输入公历出生年月日时
2. 点击"开始推算"
3. 等待分析结果缓慢浮现
4. 查看MBTI性格类型、五行能量分布等详细分析

## 核心功能

- **实时八字转换**: 自动将公历转换为农历八字
- **能量物理引擎**: 基于干支生克关系计算能量分布
- **格局辨析**: 自动识别命局格局和强弱状态
- **用神裁定**: 根据格局喜忌推荐最适合的用神

## 设计亮点

- **水墨动画**: 结果像墨水滴在纸上一样缓慢浮现
- **极细线条**: 输入框只有底部细线，聚焦时优雅扩展
- **渐进式展示**: 各分析结果按层次依次出现
- **响应式设计**: 完美适配移动端和桌面端
