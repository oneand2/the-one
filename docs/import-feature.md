# 决行藏AI导入功能使用说明

## 功能概述

决行藏AI现已支持导入功能，用户可以导入自己的八字、八维或六爻测算结果。当检测到导入的数据后，AI将自动切换到**算命模式**，使用专门的提示词来提供更精准的命理分析和指导。

## 核心功能

### 1. 数据导入
- **八字古典排盘**：导入完整的八字信息，包括：
  - 四柱（年月日时）
  - 日主强弱分析
  - 用神、忌神
  - 十神比例
  - 天干比例
  - 八字推导的MBTI
  - 能量分布（八维功能）

- **荣格八维测试**：导入八维测试结果，包括：
  - MBTI人格类型
  - 八维认知功能得分（Se, Si, Ne, Ni, Te, Ti, Fe, Fi）

- **六爻占卜**：导入六爻测算结果，包括：
  - 所问之事
  - 卦象信息（本卦、变卦）
  - 卦辞、爻辞
  - 之前的AI解卦结果

### 2. MBTI对比分析

当用户同时导入八字和八维测试结果时，AI会自动进行MBTI对比分析：

- **八字推导的MBTI**：基于出生八字推算的人格倾向（先天命理）
- **八维测试的MBTI**：通过心理测试得出的人格类型（后天发展）

如果两者存在差异，AI会温和地解释可能的原因：
- 先天命理倾向 vs 后天发展倾向
- 内在本性 vs 外在表现
- 理想自我 vs 现实自我
- 成长环境和经历对人格的塑造影响

### 3. 算命模式

当检测到导入数据后，AI将自动切换到算命模式，特点包括：

- **命理为基**：充分利用导入的命理数据进行分析
- **见微知著**：从命盘中洞察用户的本性、倾向、优势和挑战
- **古今融合**：将传统命理学与现代心理学结合
- **因材施教**：根据命理特征给出个性化建议

## 使用步骤

### 步骤1：导入数据

1. 在决行藏AI页面，点击顶部的**"导入"**按钮
2. 在弹出的对话框中，选择要导入的记录：
   - 可以单独导入八字、八维或六爻
   - 也可以同时导入多种数据
3. 点击**"确认导入"**完成导入

### 步骤2：开始对话

导入完成后，导入按钮会变为深色，表示已有导入数据。此时：

1. 输入您的问题或想要咨询的事项
2. AI会根据您导入的命理信息，提供更有针对性的分析和建议
3. 如果同时导入了八字和八维数据，AI会主动分析两者的MBTI差异

### 步骤3：清空导入（可选）

如果想清空已导入的数据：

1. 再次点击**"导入"**按钮
2. 在对话框底部点击**"清空导入"**按钮
3. 确认后，AI将恢复到普通模式

## 技术细节

### 数据处理流程

1. **数据提取**：从数据库记录中提取完整的测算信息
2. **数据重算**：对八字数据进行重新计算，确保数据完整性
3. **上下文构建**：将导入的数据格式化为结构化的上下文信息
4. **提示词切换**：自动切换到算命模式的系统提示词
5. **AI分析**：AI基于命理数据和用户问题进行综合分析

### 数据安全

- 所有导入的数据仅在当前会话中使用
- 不会永久存储或泄露给第三方
- 用户可以随时清空导入的数据

## 最佳实践

### 建议1：完整导入
为了获得最全面的分析，建议同时导入八字和八维测试结果。这样AI可以从多个维度理解您的特质。

### 建议2：具体提问
导入数据后，可以提出更具体的问题，例如：
- "根据我的八字，现在适合换工作吗？"
- "我的八字MBTI和测试MBTI不一样，这说明什么？"
- "基于我的命理特征，如何改善人际关系？"

### 建议3：定期更新
随着时间推移和人生经历的变化，建议定期重新测试八维并导入最新数据，以获得更准确的分析。

## 常见问题

**Q: 为什么八字推导的MBTI和测试的MBTI不一样？**
A: 这是正常现象。八字反映的是先天命理倾向，而测试反映的是后天发展状态。差异可能源于成长环境、人生经历、自我认知等多种因素。

**Q: 导入数据后可以改变吗？**
A: 可以。您随时可以重新导入不同的记录，或清空当前导入的数据。

**Q: 不导入数据也可以使用吗？**
A: 当然可以。不导入数据时，AI会使用普通模式的提示词，提供通用的智慧指引。

**Q: 导入的数据会被保存吗？**
A: 导入的数据仅在当前会话中临时使用，不会永久保存。每次刷新页面后需要重新导入。
